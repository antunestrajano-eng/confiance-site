# Site estático Intersolis

Pasta criada em `Desktop/intersolis-site/`.

## Como visualizar

Abra `index.html` no navegador ou sirva com qualquer servidor estático (GitHub Pages, Netlify, etc.).

## Favicon

Incluído `favicon.svg` (sol no estilo da marca). Se preferir o PNG que você exportou, renomeie para `favicon.png` e no `<head>` troque:

`<link rel="icon" href="favicon.svg" ...>` → `href="favicon.png"` e `type="image/png"`.

## Logo

O cabeçalho usa um **SVG inline** inspirado no ícone solar. Se quiser a logo oficial em imagem, substitua o bloco `.logo` por `<img src="assets/logo.png" alt="Intersolis">`.

## Portfólio (fotos)

Sim: para um grid com **fotos reais**, envie as imagens (ou coloque em `assets/projetos/` com nomes como `projeto-01.jpg`) e avise — aí trocamos os placeholders por `<img>` com legendas.

## Rodapé Instagram

O link do Instagram no rodapé está como `instagram.com/intersolis`. Ajuste para o @ oficial se for diferente.

## Conteúdo

Textos e números foram alinhados ao site público [intersolis.com.br](https://intersolis.com.br/). Depoimentos apontam para os reels/posts que você enviou.
